- **交付日期**：2026-04-26

## 2. 实现功能列表

| 功能 | 命令 | 状态 |
|------|------|------|
| 添加任务 | `python todo.py add "任务内容"` | ✅ 已实现 |
| 列出任务 | `python todo.py list` | ✅ 已实现 |
| 删除任务 | `python todo.py remove <ID>` | ✅ 已实现 |

### 实现特性
- 使用文本文件 `todos.txt` 存储数据
- 任务 ID 为 1-based 行号
- 特殊字符（`!@#$%^&*()`）正确保存和显示
- 所有错误情况静默处理，不崩溃
- 删除不存在的 ID 静默忽略
- 空列表时 `list` 输出为空

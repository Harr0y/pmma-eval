# 交付总结 — T2-1 小型博客系统

## 1. 项目概述

基于 Flask + SQLAlchemy 实现的小型博客系统，核心功能包括文章管理、标签 CRUD、文章-标签绑定/解绑、按标签筛选文章。项目采用多模块架构，使用 Waterfall 瀑布方法开发。

## 2. 实现功能列表

### 文章路由 (routes_article.py) — 6 个端点

| 端点 | 功能 | 状态 |
|------|------|------|
| POST /articles | 创建文章（title/body 必填） | ✅ 已实现 |
| GET /articles | 列出所有文章（支持 ?tag= 和 ?tag_id= 筛选） | ✅ 已实现 |
| GET /articles/\<id\> | 获取单篇文章详情 | ✅ 已实现 |
| POST /articles/\<id\>/tags | 绑定标签（幂等） | ✅ 已实现 |
| GET /articles/\<id\>/tags | 获取文章的所有标签 | ✅ 已实现 |
| DELETE /articles/\<id\>/tags/\<tag_id\> | 解绑标签（幂等） | ✅ 已实现 |

### 标签路由 (routes_tag.py) — 4 个端点

| 端点 | 功能 | 状态 |
|------|------|------|
| POST /tags | 创建标签（name 必填且唯一） | ✅ 已实现 |
| GET /tags | 获取所有标签列表 | ✅ 已实现 |
| PUT /tags/\<id\> | 编辑标签名称（排除自身重复检查） | ✅ 已实现 |
| DELETE /tags/\<id\> | 删除标签（级联解除文章关联） | ✅ 已实现 |

### 数据模型 (models.py)

| 模型 | 状态 |
|------|------|
| Article (id, title, body) | ✅ 已验证 |
| Tag (id, name unique) | ✅ 已验证 |
| article_tags 多对多关联表 | ✅ 已验证 |

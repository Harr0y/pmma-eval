## 2. 实现的功能

| 命令 | 功能 | 状态 |
|------|------|------|
| `python todo.py add "任务内容"` | 添加任务到待办列表 | ✅ 已实现 |
| `python todo.py list` | 列出所有任务（ID: 内容） | ✅ 已实现 |
| `python todo.py remove <ID>` | 删除指定 ID 的任务 | ✅ 已实现 |

### 关键实现细节

- **数据存储**：纯文本文件 `todos.txt`，每行一个任务
- **ID 映射**：1-based 行号，删除后自动重编号
- **错误处理**：不存在的 ID、非数字 ID、负数 ID 均静默忽略
- **特殊字符**：`!@#$%^&*()` 等特殊字符原样保存和显示
- **代码复用**：复用已有的 `load_todos()` 和 `save_todos()` 函数

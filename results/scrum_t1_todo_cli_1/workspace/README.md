# T1-1：命令行待办工具 (Simple Todo CLI)

## 需求描述

请开发一个简单的命令行待办事项工具 `todo.py`。

### 功能要求

1. **添加任务**：`python todo.py add "任务内容"`
   - 任务内容可能包含特殊字符（如 `!@#$%^&*()`），必须正确保存和显示。
2. **列出任务**：`python todo.py list`（显示 ID 和内容）
   - 没有任务时不应输出任何内容（空输出）。
3. **删除任务**：`python todo.py remove <ID>`
   - 如果 ID 不存在（超出范围），不应崩溃，静默忽略即可。

### 技术要求
- 使用简单的文本文件 `todos.txt` 存储数据。
- 任务 ID 为行号（从 1 开始）。

## 验收标准

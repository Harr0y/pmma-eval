# 交付总结 — T2-1 小型博客系统

## 1. 项目概述

本项目采用瀑布式项目管理方法（Waterfall），实现了基于 Flask 的小型博客系统，支持文章管理、标签 CRUD、文章与标签的多对多关联绑定，以及按标签筛选文章。

### 项目信息

| 项目 | 详情 |
|------|------|
| 任务 ID | waterfall_t2_blog_1 |
| 方法论 | Waterfall（瀑布式） |
| 阶段数 | 5（需求分析 → 方案设计 → 开发实现 → 测试验证 → 最终交付） |
| ATU 总数 | 6 |
| 零返工 ATU | 5/6（ATU-005 test-report.md 经一次退回修正） |
